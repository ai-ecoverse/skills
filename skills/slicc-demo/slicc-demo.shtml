SLICC

An AI agent that lives in your browser.

Click any concept below to go deeper — the AI will explain it to you.

Anatomy

The Cone
The main AI brain. Talks to you, remembers everything, and conducts the orchestra.
Scoops
Isolated sub-agents that do the heavy lifting — in parallel, sandboxed, and disposable.
Sprinkles
Interactive UI panels — dashboards, editors, and yes, this very screen you're reading.
Licks
Events that wake agents up — webhooks, cron jobs, or you pressing a button.
Skills
Markdown playbooks that teach agents new tricks without a single line of server code.
The Shell
UNIX commands, git, browsers — all wired in. If you can script it, SLICC can run it.
Trays
Remote runtimes. Run agents in the cloud or on another machine, browser included.
Inline Sprinkles
Mini UI cards that appear right inside the chat. Buttons, tables, pickers — no sidebar needed.
Electron App
SLICC as a desktop app. Full browser automation, local file access, no extension required.
Teleportation
Log in once on your screen. SLICC captures the session and beams it to an agent running elsewhere.
Secret Sauce
Reverse-engineer any web app's API. One HAR file. One skill. Zero UI friction forever.
The Wildcard
Surprise me. The cone picks two or three concepts at random and builds something unexpected.
